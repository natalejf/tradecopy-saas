📘 Quick Setup Guide
1

Add your Master account

Go to Accounts → Add Account. Choose role Master. Fill in your MT5 login number and server name (not the password — never stored in cloud). You'll get an EA Token.

2

Install the Master EA in MT5

Download TradeCopy_Master.ex5 from the EA tab. Copy it to your MT5 MQL5/Experts folder. Compile it. Drag it onto any chart. Set ServerURL to your server and paste your EAToken. Make sure to allow WebRequest for your server URL in MT5 settings.

3

Add Follower accounts

Repeat step 1 for each follower account, choosing role Follower. Each gets its own unique EA token. Install TradeCopy_Follower.ex5 on each follower MT5.

4

Create a Copy Group

Go to Copy Groups → New Group. Select your master account and add followers. Trades from the master will now be automatically replicated to all followers in the group.

5

Configure lot sizing (optional)

Edit each follower account to set lot_multiplier, fixed_lot, or max_lot. You can also enable reverse copying or add symbol mappings if your brokers use different symbol names (e.g. EURUSD vs EURUSDm).

6

Monitor in real-time

Go to Dashboard to see the live event feed. Every trade open/close/modify on the master will appear here and be automatically copied to all linked followers.
